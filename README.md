# Bumerange CDN

Centralized brand assets served via Cloudflare Pages.

## Usage

```
https://cdn.bumerange.com/logos/bumerange-icon.svg
https://cdn.bumerange.com/logos/bumerange-horizontal.svg
https://cdn.bumerange.com/logos/bumerange-stacked.svg
https://cdn.bumerange.com/favicons/favicon.svg
https://cdn.bumerange.com/favicons/favicon.ico
```

## Structure

```
/logos          - SVG logo variants
/favicons       - Favicon assets (ico, svg, png)
/brand          - Brand guidelines and colors
```

## Deployment

Auto-deploys to Cloudflare Pages on push to `main`.
